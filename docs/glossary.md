# Glossary

- GIS, Geographic Information System
- Raster data, uses a grid of equally sized cells or pixels to represent continuous, gradually changing geographic phenomena. This is analogous to a JPEG image.
- Vector data, uses \(x, y\) coordinates to represent geographic features with distinct, hard boundaries.
  - Points: Zero-dimensional features (e.g., specific locations, ATMs, or weather stations).
  - Lines (Arcs): One-dimensional features with a start and end (e.g., roads, rivers, or pipelines).
  - Polygons: Two-dimensional enclosed areas (e.g., land parcels, lakes, or zip code boundaries).
- Attribute data, the descriptive, tabular information linked to your spatial data (Vector or Raster).
